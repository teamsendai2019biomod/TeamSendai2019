Improved design

RHCR is a mechanism that can extend and shrink the linear complex while maintaining the cross-linking point. In RHCR the elongation reaction starts from the state Q0, and the shrinking reaction ends at Q1. This is because crosslinking strand A does not have a toehold to initiate four-way branch migration with R (fig. 1)

fig.1 Shrinking reaction stops at Q'1

Therefore, if a toehold (light blue domain) reacts with R is attached to A, the shrinking reaction occurs perfectly(fig. 2). In such a design, since the restriction site emerges at A, unintended nicking reaction may occur. However, there is no effect on the extending reaction.

fig.2 A with toehold for four-way branch migration

The version shown in fig.2 can shrink until it reaches the state Q0. Since the elongation by one hairpin is about 13 nanometers, the impact of not returning to Q0 is negligible. In fact, we decided that not attach a new toehold to DNA A because we put priority in the extending reaction. However, this improvement that makes the whole reaction recycling, it is valuable if this mechanism is used for nanorobot in the future.

Future application of RHCR

Reversible Hybridization Chain Reaction (RHCR) makes it possible to build new systems. For instance, A system similar to action comet can be applied for Drug Delivery System (DDS), which can move freely within our body. The mechanism can be also applied to the crosslink of the hydrogel, which can swell and shrink reversibly(fig3). The hydrogel that can change its size in response to signals from outside is expected to be utilized as soft robots. The hydrogel can be formed into any shape and this materials will be used as a shape memory gel.

Another way to make use of this system is lazy tongs(or magic hand)-like device. When one end of RHCR complex is stuck on a base, it is used to fetch something a littl.
